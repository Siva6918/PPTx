Line Clipping Algorithms

Cohen–Sutherland & Liang–Barsky

Introduction

  • Line clipping removes invisible parts of lines.
  • Uses a defined rectangular clipping window.
  • Maintains visual accuracy while rendering.
  • Improves rendering performance drastically.

Types of Cases

  • ✅ Line completely inside → Accept
  • ❌ Line completely outside → Reject
  • ✂️ Line partially inside → Clip

Cohen–Sutherland Concept

  • Divides coordinate space into 9 regions.
  • Assigns 4-bit region codes (Outcodes).
  • Each endpoint of a line gets an Outcode.
  • Quickly assesses trivial accept/reject.

Region Codes (Outcodes)

Every region receives a 4-bit binary code (TBRL)

Top → 1000
Bottom → 0100
Right → 0010
Left → 0001
Inside Window → 0000

Cohen-Sutherland: Working Steps

  1. Assign 4-bit region codes to both line endpoints.
  2. If both codes are 0000 → Trivially Accept.
  3. If Bitwise AND of codes ≠ 0000 → Trivially Reject.
  4. Else → Find intersection point with window boundary and clip. Repeat.

Cohen-Sutherland: Pros & Cons

Advantages

  • Simple and easy to understand.
  • Extremely fast for basic cases (trivial accept/reject).

Disadvantages

  • Requires recursive intersection checks.
  • More calculations needed for lines crossing multiple boundaries.

Liang–Barsky Concept

  • Uses the parametric equation of a line.
  • Focuses on a parameter t (0 to 1).
  • Analyzes differences between point coordinates and window edges.
  • Significantly more efficient than Cohen–Sutherland.

Parametric Line Equation

A line segment can be defined parametrically:

x = x₁ + t × (x₂ − x₁)
y = y₁ + t × (y₂ − y₁)

Valid t ranges from 0 to 1

Liang-Barsky: Working Steps

  1. Define directional values p (line direction) and q (distance to edge).
  2. Calculate intersection parameter t = q / p for all 4 edges.
  3. Find valid t1 (line entering) and t2 (line exiting).
  4. If t1 < t2, calculate new endpoints. Else, reject.

Liang-Barsky: Pros & Cons

Advantages

  • Fewer intersection calculations needed.
  • Highly efficient math execution.
  • Checks all edges simultaneously.

Disadvantages

  • Mathematical theory is slightly more complex to grasp.

Conclusion

  • Line Clipping is vital for saving processing power and rendering time.
  • Cohen–Sutherland provides an intuitive, simple approach.
  • Liang–Barsky offers a heavily optimized, mathematical approach.
